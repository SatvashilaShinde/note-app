📝 Notes App

A simple and interactive Notes App built using HTML, CSS, and JavaScript.
This app allows users to create, edit, and delete notes, with data saved in the browser using localStorage.

🚀 Features
Create new notes instantly
Edit notes directly in the browser
Delete notes with a single click
Data persistence using localStorage (notes remain after refresh)
Simple and user-friendly interface

🛠️ Technologies Used
HTML
CSS
JavaScript (DOM Manipulation & Local Storage)

💡 How It Works
Notes are created dynamically using JavaScript
Each note is editable using the contenteditable attribute
Notes are saved in localStorage
Data is automatically updated when typing or deleting notes
